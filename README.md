# tweeme-ios
Ultimate tweet-only Twitter tool

## Jak uruchomić aplikację?

1. Zainstaluj Tuist

`bash <(curl -Ls https://install.tuist.io)`

2. Przejdź w Terminalu do folderu głównego
3. Wywołaj kolejno

`tuist up`

`tuist focus`
